Добро пожаловать в многофункциональный разборщик ARINC!
-----------------------------------------------------------------
Перед первым выполнением разбора ARINC, рекомендуется выполнить проверку на порядок нумерации точек.
______________________________________________________________________
Результат выполнения программы находится в следующих папках:

1) Запросы в БД MySQL - /MULTI_ARINC/1.SINTEZ/1.Запросы
-----------------------------------------------------------------
2) Слои картографии SLD - /MULTI_ARINC/1.SINTEZ/2.Картография
-----------------------------------------------------------------
3) Таблицы и файлы с информацией о новых и старых трассах для диспечерского состава - 
/MULTI_ARINC/1.SINTEZ/3.Таблицы
-----------------------------------------------------------------
4) Файл с точками, которые имеют одинаковые названия и разные координаты, попавшие в зону полигона - 
/MULTI_ARINC/1.SINTEZ/4.duplicates.txt
-----------------------------------------------------------------
5) Файл для экспорта в программу КОРИНФ на места ЗКП - /MULTI_ARINC/2.KORINF/export.all
-----------------------------------------------------------------
6) Картография для индикатора на позиции ОРЛ-А - /MULTI_ARINC/3.ORL-A/map_дата.mp
______________________________________________________________________

Изменение параметров (форма и размер полигона, расстояние между десигнаторами) можно сделать в конфигурационном файле.
